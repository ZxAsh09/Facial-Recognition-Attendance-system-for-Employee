# Facial-Recognition-Attendance-system-for-Employee

AI-powered attendance management system using Flask and OpenCV.

## Features

- Real-time face recognition
- Attendance automation
- Flask web dashboard
- LAN-based access
- Employee registration
- Reports and analytics
- Admin panel
- Live webcam feed

## Technologies Used

- Python
- Flask
- OpenCV
- HTML/CSS
- JavaScript
- Pandas

## Run Project

```bash
pip install -r requirements.txt
python app.py
